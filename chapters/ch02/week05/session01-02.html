2.2 Atomic Number, Isotopes, Drawbacks of Rutherford Model

KEY CONCEPTS FROM TODAY:
  • Atomic number (Z) = number of protons; defines the element
  • Mass number (A) = protons + neutrons
  • Isotopes: same Z, different A (same element, different neutron count)
  • Rutherford's model could not explain atomic stability, spectra, or atomic number — motivating Bohr's model
