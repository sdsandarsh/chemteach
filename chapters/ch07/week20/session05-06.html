6.12 Buffer Solutions

✓ SESSION COMPLETE

KEY CONCEPTS FROM TODAY:
  • Buffer solution: weak acid + conjugate base (or weak base + conjugate acid) mixture
  • Resists pH change by absorbing added H+ or OH- via the common ion equilibrium
  • Henderson-Hasselbalch equation: pH = pKa + log([conjugate base]/[weak acid])
  • Buffer capacity is finite — exceeding it causes rapid pH change
  • Blood pH is maintained within a narrow range by natural buffer systems
